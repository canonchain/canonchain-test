
jar包使用方式 ：
	
    1. 打开cmd或者gitbash或者cygwin；

    2. 调用方式 java -jar contentCapture.jar test.txt content

         test.txt : 需要被抓取内容的目标文件  

  	 content : 需要抓取匹配的内容

exe使用方式 ：
	
    1. 打开cmd或者gitbash或者cygwin；

    2. 在相应的目录下调用方式：

         contentCapture.exe test.txt content

             test.txt : 需要被抓取内容的目标文件  

             content : 需要抓取匹配的内容
